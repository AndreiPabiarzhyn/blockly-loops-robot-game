#  Blockly Loops Robot Game

Интерактивная игра для обучения основам программирования с помощью Blockly.

Игрок управляет роботом, собирает кристаллы  и проходит уровни, используя алгоритмы и циклы.

---

## Демо

👉 https://andreipabiarzhyn.github.io/blockly-loops-robot-game/

---

## Чему учит игра

- последовательность действий
- алгоритмическое мышление
- использование циклов (loops)
- оптимизация решений

---

## Как играть

1. Перетаскивай блоки справа
2. Собирай программу
3. Нажми ▶ Запустить
4. Робот выполнит команды

---

## Механики

- 🤖 Робот перемещается по сетке
- 💎 Нужно собрать все кристаллы
- 🚧 Стены — проигрыш при столкновении
- 🌀 Портал — выход с уровня
- 🔁 Поддержка циклов (repeat)
